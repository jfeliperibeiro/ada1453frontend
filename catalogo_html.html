Estrutura do Documento

<html>

Formato: <html>...</html>

Descrição:

O elemento <html> representa a raiz de um documento HTML ou XHTML. Todos os outros elementos devem ser descendentes deste elemento.

Exemplo de Uso:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Olá, Mundo!</h1>
    </body>
</html>
<head>

Formato: <head>...</head>

Descrição:

O elemento <head> contém metadados sobre o documento, como seu título, links para folhas de estilo, scripts e outras informações que não são visíveis no corpo da página.

Exemplo de Uso:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
    <link rel="stylesheet" href="style.css">
    <script src="script.js" defer></script>
</head>
<body>

Formato: <body>...</body>

Descrição:

O elemento <body> representa o conteúdo principal de um documento HTML. Tudo o que é visível para o usuário (texto, imagens, vídeos, etc.) é colocado dentro desta tag.

Exemplo de Uso:

<body>
    <header>
        <h1>Bem-vindo!</h1>
    </header>
    <main>
        <p>Este é o conteúdo principal da página.</p>
    </main>
    <footer>
        <p>© 2023 Minha Empresa</p>
    </footer>
</body>

Metadados

<title>

Formato: <title>...</title>

Descrição:

O elemento <title> define o título do documento, que é exibido na barra de título do navegador ou na aba da página. É crucial para SEO e acessibilidade.

Exemplo de Uso:

<head>
    <title>Minha Home Page Incrível</title>
</head>
<meta>

Formato: <meta> (self-closing)

Descrição:

O elemento <meta> fornece metadados sobre o documento HTML. Não tem tag de fechamento e é usado para especificar o conjunto de caracteres, a viewport, descrição da página, palavras-chave, etc.

Exemplo de Uso:

<head>
    <meta charset="UTF-8">
    <meta name="description" content="Uma página sobre tags HTML.">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<link>

Formato: <link> (self-closing)

Descrição:

O elemento <link> especifica a relação entre o documento atual e um recurso externo, como uma folha de estilo ou um ícone.

Exemplo de Uso:

<head>
    <link rel="stylesheet" href="styles.css">
    <link rel="icon" href="favicon.ico" type="image/x-icon">
</head>
<style>

Formato: <style>...</style>

Descrição:

O elemento <style> é usado para incorporar folhas de estilo CSS diretamente no documento HTML.

Exemplo de Uso:

<head>
    <style>
        body {
            font-family: Arial, sans-serif;
            color: #333;
        }
        h1 {
            color: blue;
        }
    </style>
</head>
<script>

Formato: <script>...</script>

Descrição:

O elemento <script> é usado para incorporar ou referenciar scripts executáveis, geralmente JavaScript. Pode ser colocado no <head> ou no <body>.

Exemplo de Uso:

<body>
    <p id="demo"></p>

    <script>
        document.getElementById("demo").innerHTML = "Hello JavaScript!";
    </script>

    <script src="meu-script.js" defer></script>
</body>

Conteúdo de Texto

<h1> a <h6>

Formato: <h1>...</h1>, <h2>...</h2>, etc.

Descrição:

Os elementos <h1> a <h6> representam seis níveis de títulos de seção. <h1> é o mais importante e <h6> é o menos importante. Essenciais para a estrutura semântica de um documento.

Exemplo de Uso:

<h1>Título Principal da Página</h1>
<h2>Subtítulo da Seção</h2>
<h3>Título de um Artigo</h3>
<p>

Formato: <p>...</p>

Descrição:

O elemento <p> representa um parágrafo de texto. É a tag mais comum para blocos de texto em HTML.

Exemplo de Uso:

<p>Este é o primeiro parágrafo do meu texto. Ele contém informações importantes.</p>
<p>Este é o segundo parágrafo. Cada parágrafo é um bloco separado.</p>
<a>

Formato: <a>...</a>

Descrição:

O elemento <a> (âncora) cria um hyperlink para outros documentos, partes do mesmo documento ou para um endereço de e-mail. O atributo principal é href.

Exemplo de Uso:

<p>Visite o <a href="https://www.w3.org/" target="_blank">site do W3C</a> para mais informações.</p>
<p><a href="#top">Voltar ao topo</a></p>
<br>

Formato: <br> (self-closing)

Descrição:

O elemento <br> produz uma quebra de linha. É útil quando uma quebra de linha é semanticamente importante, como em endereços ou poesia.

Exemplo de Uso:

<p>Rua Exemplo, 123<br>
Cidade, Estado<br>
CEP 12345-678</p>
<hr>

Formato: <hr> (self-closing)

Descrição:

O elemento <hr> representa uma quebra temática entre parágrafos de conteúdo (por exemplo, uma mudança de cena em uma história ou uma mudança de tópico). É renderizado como uma linha horizontal por padrão.

Exemplo de Uso:

<p>Fim da introdução.</p>
<hr>
<p>Início do conteúdo principal.</p>
<strong>

Formato: <strong>...</strong>

Descrição:

O elemento <strong> indica que seu conteúdo tem grande importância, seriedade ou urgência. Navegadores geralmente o renderizam em negrito.

Exemplo de Uso:

<p><strong>Atenção:</strong> Esta mensagem é crucial.</p>
<em>

Formato: <em>...</em>

Descrição:

O elemento <em> (ênfase) marca o texto que deve ser enfatizado. Geralmente é renderizado em itálico pelos navegadores.

Exemplo de Uso:

<p>Eu <em>realmente</em> gosto de HTML.</p>
<small>

Formato: <small>...</small>

Descrição:

O elemento <small> representa comentários laterais e pequenas impressões, como direitos autorais, avisos legais ou texto de rodapé. Geralmente renderizado com uma fonte menor.

Exemplo de Uso:

<p><small>© 2023 Todos os direitos reservados.</small></p>
<code>

Formato: <code>...</code>

Descrição:

O elemento <code> exibe um fragmento de código de programação. Conteúdo é exibido em uma fonte monoespaçada por padrão.

Exemplo de Uso:

<p>A função <code>console.log()</code> é usada para depuração.</p>
<pre>

Formato: <pre>...</pre>

Descrição:

O elemento <pre> representa texto pré-formatado. O texto dentro dele é exibido em uma fonte monoespaçada e mantém espaços em branco e quebras de linha exatamente como digitados.

Exemplo de Uso:

<pre>
function saudacao() {
    console.log("Olá, mundo!");
}
saudacao();
</pre>
<blockquote>

Formato: <blockquote>...</blockquote>

Descrição:

O elemento <blockquote> indica que o texto contido é uma citação estendida de outra fonte. Geralmente é indentado pelos navegadores.

Exemplo de Uso:

<blockquote cite="https://www.example.com/quote">
    <p>A única forma de fazer um grande trabalho é amar o que você faz.</p>
    <footer>— Steve Jobs</footer>
</blockquote>
<q>

Formato: <q>...</q>

Descrição:

O elemento <q> representa uma citação curta inline. Navegadores geralmente adicionam aspas automaticamente.

Exemplo de Uso:

<p>Ele disse <q cite="https://www.example.com/speech">Eu voltarei amanhã.</q> antes de sair.</p>
<abbr>

Formato: <abbr>...</abbr>

Descrição:

O elemento <abbr> representa uma abreviação ou um acrônimo. O atributo title pode ser usado para fornecer a expansão completa.

Exemplo de Uso:

<p>O <abbr title="HyperText Markup Language">HTML</abbr> é a linguagem padrão da web.</p>
<address>

Formato: <address>...</address>

Descrição:

O elemento <address> indica informações de contato para seu <article> ou para todo o documento.

Exemplo de Uso:

<address>
  Escrito por <a href="mailto:webmaster@example.com">John Doe</a>.<br>
  Visite-nos em:<br>
  Exemplo.com<br>
  Rua Exemplo, 123<br>
  Brasil
</address>
<cite>

Formato: <cite>...</cite>

Descrição:

O elemento <cite> representa o título de um trabalho criativo (por exemplo, um livro, uma peça, uma música, um filme, uma pintura, uma escultura, etc.). Geralmente renderizado em itálico.

Exemplo de Uso:

<p>Meu livro favorito é <cite>O Pequeno Príncipe</cite>.</p>
<del>

Formato: <del>...</del>

Descrição:

O elemento <del> representa um intervalo de texto que foi excluído de um documento. Geralmente é renderizado com um risco no meio (tachado).

Exemplo de Uso:

<p>Preço: <del>R\$ 100,00</del> R\$ 75,00</p>
<ins>

Formato: <ins>...</ins>

Descrição:

O elemento <ins> representa um intervalo de texto que foi adicionado a um documento. Geralmente é renderizado sublinhado.

Exemplo de Uso:

<p>Status: <del>Pendente</del> <ins>Concluído</ins></p>
<mark>

Formato: <mark>...</mark>

Descrição:

O elemento <mark> representa o texto que é marcado ou destacado por sua relevância em um determinado contexto. Geralmente é renderizado com um fundo amarelo.

Exemplo de Uso:

<p>Pesquise por <mark>HTML5</mark> para aprender mais.</p>
<sub>

Formato: <sub>...</sub>

Descrição:

O elemento <sub> exibe seu conteúdo como subscrito (abaixo da linha base do texto) e com uma fonte menor.

Exemplo de Uso:

<p>A fórmula da água é H<sub>2</sub>O.</p>
<sup>

Formato: <sup>...</sup>

Descrição:

O elemento <sup> exibe seu conteúdo como sobrescrito (acima da linha base do texto) e com uma fonte menor.

Exemplo de Uso:

<p>2<sup>3</sup> é igual a 8.</p>

Listas

<ul>

Formato: <ul>...</ul>

Descrição:

O elemento <ul> representa uma lista não ordenada de itens. Os itens são geralmente marcados com bullet points.

Exemplo de Uso:

<ul>
    <li>Maçãs</li>
    <li>Bananas</li>
    <li>Laranjas</li>
</ul>
<ol>

Formato: <ol>...</ol>

Descrição:

O elemento <ol> representa uma lista ordenada de itens. Os itens são geralmente numerados.

Exemplo de Uso:

<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
    <li>Terceiro passo</li>
</ol>
<li>

Formato: <li>...</li>

Descrição:

O elemento <li> representa um item de lista dentro de uma lista ordenada (<ol>) ou não ordenada (<ul>).

Exemplo de Uso:

<ul>
    <li>Item de Lista</li>
</ul>
<dl>

Formato: <dl>...</dl>

Descrição:

O elemento <dl> (description list) representa uma lista de descrições, onde cada termo (<dt>) pode ter uma ou mais descrições (<dd>).

Exemplo de Uso:

<dl>
    <dt>HTML</dt>
    <dd>Linguagem de Marcação de Hipertexto.</dd>
    <dt>CSS</dt>
    <dd>Folhas de Estilo em Cascata.</dd>
</dl>
<dt>

Formato: <dt>...</dt>

Descrição:

O elemento <dt> (description term) representa um termo em uma lista de descrições (<dl>).

Exemplo de Uso:

<dl>
    <dt>Termo</dt>
    <dd>Descrição.</dd>
</dl>
<dd>

Formato: <dd>...</dd>

Descrição:

O elemento <dd> (description details) representa a descrição ou definição de um termo em uma lista de descrições (<dl>).

Exemplo de Uso:

<dl>
    <dt>Termo</dt>
    <dd>Descrição.</dd>
</dl>

Links e Mídia

<img>

Formato: <img> (self-closing)

Descrição:

O elemento <img> incorpora uma imagem no documento. Os atributos src e alt são obrigatórios.

Exemplo de Uso:

<img src="logo.png" alt="Logo da empresa" width="150" height="50">
<picture>

Formato: <picture>...</picture>

Descrição:

O elemento <picture> contém zero ou mais elementos <source> e um elemento <img> para oferecer versões diferentes de uma imagem para diferentes cenários de dispositivo/resolução.

Exemplo de Uso:

<picture>
    <source srcset="large.webp" media="(min-width: 800px)">
    <source srcset="medium.webp" media="(min-width: 400px)">
    <img src="small.jpg" alt="Exemplo de imagem responsiva">
</picture>
<source>

Formato: <source> (self-closing)

Descrição:

O elemento <source> especifica múltiplos recursos de mídia para <picture>, <audio> e <video>, permitindo que o navegador escolha o formato mais adequado.

Exemplo de Uso:

<video controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Seu navegador não suporta a tag de vídeo.
</video>
<audio>

Formato: <audio>...</audio>

Descrição:

O elemento <audio> é usado para incorporar conteúdo de áudio em um documento HTML. Atributos como controls, autoplay e loop podem ser usados.

Exemplo de Uso:

<audio controls autoplay loop>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Seu navegador não suporta a tag de áudio.
</audio>
<video>

Formato: <video>...</video>

Descrição:

O elemento <video> é usado para incorporar conteúdo de vídeo em um documento HTML. Atributos como controls, autoplay, loop, width, height e poster são comuns.

Exemplo de Uso:

<video controls width="320" height="240" poster="thumb.jpg">
    <source src="movie.mp4" type="video/mp4">
    Seu navegador não suporta a tag de vídeo.
</video>
<iframe>

Formato: <iframe>...</iframe>

Descrição:

O elemento <iframe> é usado para incorporar outro documento HTML dentro do documento atual (em um "quadro" embutido). É frequentemente usado para incorporar mapas, vídeos do YouTube, etc.

Exemplo de Uso:

<iframe src="https://www.example.com" width="600" height="400" allowfullscreen></iframe>

Tabelas

<table>

Formato: <table>...</table>

Descrição:

O elemento <table> representa dados tabulares, ou seja, informações dispostas em um formato de tabela bidimensional com linhas e colunas.

Exemplo de Uso:

<table>
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>30</td>
    </tr>
</table>
<caption>

Formato: <caption>...</caption>

Descrição:

O elemento <caption> especifica o título ou legenda de uma tabela. Deve ser o primeiro elemento filho de <table>.

Exemplo de Uso:

<table>
    <caption>Dados de Vendas Mensais</caption>
    <tr>...</tr>
</table>
<thead>

Formato: <thead>...</thead>

Descrição:

O elemento <thead> agrupa o conteúdo do cabeçalho de uma tabela, contendo as linhas que definem os títulos das colunas.

Exemplo de Uso:

<table>
    <thead>
        <tr>
            <th>Produto</th>
            <th>Preço</th>
        </tr>
    </thead>
    <tbody>...</tbody>
</table>
<tbody>

Formato: <tbody>...</tbody>

Descrição:

O elemento <tbody> agrupa o conteúdo do corpo de uma tabela, contendo as linhas de dados.

Exemplo de Uso:

<table>
    <thead>...</thead>
    <tbody>
        <tr>
            <td>Monitor</td>
            <td>R\$ 800,00</td>
        </tr>
    </tbody>
</table>
<tfoot>

Formato: <tfoot>...</tfoot>

Descrição:

O elemento <tfoot> agrupa o conteúdo do rodapé de uma tabela, tipicamente usado para totais ou resumos.

Exemplo de Uso:

<table>
    <thead>...</thead>
    <tbody>...</tbody>
    <tfoot>
        <tr>
            <td>Total</td>
            <td>R\$ 1500,00</td>
        </tr>
    </tfoot>
</table>
<tr>

Formato: <tr>...</tr>

Descrição:

O elemento <tr> (table row) define uma linha de células em uma tabela.

Exemplo de Uso:

<table>
    <tr>
        <td>Célula 1</td>
        <td>Célula 2</td>
    </tr>
</table>
<th>

Formato: <th>...</th>

Descrição:

O elemento <th> (table header) define uma célula de cabeçalho em uma tabela. Serve para identificar as colunas ou linhas.

Exemplo de Uso:

<tr>
    <th>Nome</th>
    <th>Email</th>
</tr>
<td>

Formato: <td>...</td>

Descrição:

O elemento <td> (table data) define uma célula de dados padrão em uma tabela.

Exemplo de Uso:

<tr>
    <td>Maria</td>
    <td>maria@example.com</td>
</tr>

Formulários

<form>

Formato: <form>...</form>

Descrição:

O elemento <form> representa uma seção do documento que contém controles interativos para enviar informações a um servidor web. Atributos como action e method são essenciais.

Exemplo de Uso:

<form action="/submit_data" method="post">
    <label for="name">Nome:</label>
    <input type="text" id="name" name="user_name">
    <input type="submit" value="Enviar">
</form>
<input>

Formato: <input> (self-closing)

Descrição:

O elemento <input> é usado para criar controles de formulário interativos para que os usuários insiram dados. O atributo type define o tipo de controle (text, password, checkbox, radio, submit, etc.).

Exemplo de Uso:

<input type="text" name="username" placeholder="Nome de usuário" required>
<input type="password" name="password">
<input type="checkbox" name="remember_me" id="remember">
<label for="remember">Lembrar-me</label>
<input type="submit" value="Login">
<label>

Formato: <label>...</label>

Descrição:

O elemento <label> representa uma legenda para um item em uma interface de usuário. É associado a um controle de formulário pelo atributo for, que deve corresponder ao id do controle.

Exemplo de Uso:

<label for="email">Endereço de Email:</label>
<input type="email" id="email" name="user_email">
<button>

Formato: <button>...</button>

Descrição:

O elemento <button> representa um botão clicável. Ele pode ser usado dentro de um formulário ou em qualquer outro lugar do documento.

Exemplo de Uso:

<button type="submit">Enviar Formulário</button>
<button type="button" onclick="alert('Olá!');">Clique-me</button>
<textarea>

Formato: <textarea>...</textarea>

Descrição:

O elemento <textarea> representa um controle de edição de texto multi-linha. Atributos como rows e cols controlam o tamanho inicial.

Exemplo de Uso:

<label for="mensagem">Sua Mensagem:</label>
<textarea id="mensagem" name="user_message" rows="5" cols="30" placeholder="Digite aqui..."></textarea>
<select>

Formato: <select>...</select>

Descrição:

O elemento <select> representa um controle que fornece um menu de opções (<option>) para o usuário escolher.

Exemplo de Uso:

<label for="fruta">Escolha uma fruta:</label>
<select id="fruta" name="selected_fruit">
    <option value="maca">Maçã</option>
    <option value="banana">Banana</option>
    <option value="laranja" selected>Laranja</option>
</select>
<option>

Formato: <option>...</option>

Descrição:

O elemento <option> é usado para definir uma opção dentro de um elemento <select>, <optgroup> ou <datalist>.

Exemplo de Uso:

<select>
    <option value="carro">Carro</option>
</select>
