Layout e Posicionamento

display

Nome da Propriedade: display

Descrição:

A propriedade display define o tipo de caixa de exibição de um elemento. É a propriedade mais fundamental para controlar o layout de um elemento, determinando como ele se comporta em relação a outros elementos na página.

Valores Possíveis:

  • block: O elemento ocupa a largura total disponível e inicia uma nova linha (ex: div, p, h1).
  • inline: O elemento ocupa apenas a largura necessária para seu conteúdo e não inicia uma nova linha (ex: span, a, strong).
  • inline-block: Comporta-se como inline, mas aceita width, height, margin e padding como um elemento de bloco.
  • flex: Torna o elemento um flex container, permitindo o layout de seus filhos diretos (flex items) usando o modelo Flexbox.
  • grid: Torna o elemento um grid container, permitindo o layout de seus filhos diretos (grid items) usando o modelo Grid.
  • none: O elemento não é exibido e é completamente removido do fluxo do documento (como se não existisse).
  • Outros: table, table-cell, list-item, flow-root, etc.

Exemplos de Uso:

/* Elemento de bloco */
p {
    display: block;
}

/* Elemento inline */
span {
    display: inline;
}

/* Elemento inline-block */
.button {
    display: inline-block;
    width: 100px;
    height: 30px;
    padding: 5px;
}

/* Container Flexbox */
.flex-container {
    display: flex;
    justify-content: space-around;
}

/* Container Grid */
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 10px;
}

/* Esconder elemento */
.hidden-element {
    display: none;
}
position

Nome da Propriedade: position

Descrição:

A propriedade position especifica o tipo de método de posicionamento usado para um elemento. Combinada com top, bottom, left e right, ela controla o posicionamento exato.

Valores Possíveis:

  • static: Posição padrão. Elementos são posicionados de acordo com o fluxo normal do documento. top, bottom, left, right não têm efeito.
  • relative: O elemento é posicionado de acordo com o fluxo normal, mas então deslocado de sua posição normal. top, bottom, left, right funcionam. Ocupa espaço no fluxo normal.
  • absolute: O elemento é removido do fluxo normal do documento e posicionado em relação ao seu ancestral posicionado mais próximo (não static). Se não houver, é posicionado em relação ao <html>.
  • fixed: O elemento é removido do fluxo normal e posicionado em relação à viewport. Permanece no mesmo lugar mesmo quando a página é rolada.
  • sticky: Comporta-se como relative até que o elemento atinja um limite de deslocamento (definido por top, etc.) quando se torna fixed.

Exemplos de Uso:

/* Elemento relativo */
.box-relative {
    position: relative;
    top: 20px;
    left: 30px;
}

/* Elemento absoluto dentro de um pai relativo */
.parent {
    position: relative;
    height: 200px;
    border: 1px solid black;
}
.child-absolute {
    position: absolute;
    top: 0;
    right: 0;
    background-color: lightblue;
}

/* Elemento fixo */
.header-fixed {
    position: fixed;
    top: 0;
    width: 100%;
    background-color: #333;
    color: white;
    padding: 10px;
}

/* Elemento sticky */
.sidebar-sticky {
    position: sticky;
    top: 0; /* Irá fixar quando a rolagem atingir o topo */
}
float

Nome da Propriedade: float

Descrição:

A propriedade float é usada para posicionar um elemento à esquerda ou à direita de seu contêiner, permitindo que o texto e os elementos inline fluam ao redor dele. Era amplamente usada para layouts de múltiplas colunas antes do Flexbox e Grid.

Valores Possíveis:

  • left: Flutua o elemento à esquerda.
  • right: Flutua o elemento à direita.
  • none: Valor padrão. O elemento não flutua.
  • inherit: Herda o valor do elemento pai.

Exemplos de Uso:

.image-float-left {
    float: left;
    margin-right: 15px;
}

.text-content {
    /* O texto irá fluir ao redor da imagem flutuante */
}

/* Para "limpar" o float, impedindo que o conteúdo subsequente se alinhe */
.clear-float {
    clear: both; /* Ou left, ou right */
}
clear

Nome da Propriedade: clear

Descrição:

A propriedade clear especifica de que lado (ou de ambos) um elemento não pode ter elementos flutuantes. É usada para controlar o fluxo de layout após elementos flutuantes.

Valores Possíveis:

  • none: Padrão. O elemento pode flutuar de ambos os lados.
  • left: O elemento se move abaixo de qualquer elemento flutuante à esquerda.
  • right: O elemento se move abaixo de qualquer elemento flutuante à direita.
  • both: O elemento se move abaixo de qualquer elemento flutuante (à esquerda ou à direita).

Exemplos de Uso:

<div style="float: left; width: 100px; height: 100px; background: red;"></div>
<p>Este texto irá fluir ao lado do div flutuante.</p>
<div style="clear: both; background: green; height: 50px;">Este div limpa o float e aparece abaixo.</div>
Flexbox: flex-direction, justify-content, align-items

Nome da Propriedade: flex-direction

Descrição:

Define a direção principal em que os flex items são colocados no flex container (o eixo principal).

Valores Possíveis:

  • row (padrão): Da esquerda para a direita (ou direita para a esquerda em LTR/RTL).
  • row-reverse: Da direita para a esquerda (ou esquerda para a direita em LTR/RTL).
  • column: De cima para baixo.
  • column-reverse: De baixo para cima.

Exemplo de Uso:

.container-flex {
    display: flex;
    flex-direction: column; /* Itens empilhados verticalmente */
}

Nome da Propriedade: justify-content

Descrição:

Alinha os flex items ao longo do eixo principal do flex container.

Valores Possíveis:

  • flex-start: Itens alinhados no início do eixo principal.
  • flex-end: Itens alinhados no final do eixo principal.
  • center: Itens centralizados no eixo principal.
  • space-between: O primeiro item fica no início, o último no final, e o espaço restante é distribuído igualmente entre os itens.
  • space-around: O espaço é distribuído igualmente ao redor de cada item (metade do espaço nas extremidades).
  • space-evenly: O espaço é distribuído igualmente entre os itens e nas extremidades.

Exemplo de Uso:

.container-flex {
    display: flex;
    justify-content: space-between; /* Espaço entre os itens */
}

Nome da Propriedade: align-items

Descrição:

Alinha os flex items ao longo do eixo transversal (perpendicular ao eixo principal) do flex container.

Valores Possíveis:

  • flex-start: Itens alinhados no início do eixo transversal.
  • flex-end: Itens alinhados no final do eixo transversal.
  • center: Itens centralizados no eixo transversal.
  • baseline: Itens alinhados com base na linha de base de seu conteúdo.
  • stretch (padrão): Itens esticados para preencher o contêiner (respeitando min-height/max-height).

Exemplo de Uso:

.container-flex {
    display: flex;
    height: 200px; /* Para visualizar o alinhamento */
    align-items: center; /* Centraliza verticalmente os itens */
}
Grid: grid-template-columns, grid-template-rows, gap

Nome da Propriedade: grid-template-columns

Descrição:

Define os nomes e as funções de cada coluna na grade.

Valores Possíveis:

  • none: Nenhuma coluna explícita.
  • <length>: Largura fixa (ex: 100px).
  • <percentage>: Largura percentual (ex: 50%).
  • fr unit: Fração do espaço disponível (ex: 1fr, 2fr).
  • auto: A largura é determinada automaticamente.
  • min-content, max-content: Largura baseada no conteúdo.
  • repeat(): Repete um padrão (ex: repeat(3, 1fr)).

Exemplo de Uso:

.grid-container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* Três colunas: 1 parte, 2 partes, 1 parte */
    grid-template-rows: auto 100px; /* Linha automática, e uma de 100px */
}

Nome da Propriedade: grid-template-rows

Descrição:

Define os nomes e as funções de cada linha na grade.

Valores Possíveis:

  • Mesmos valores que grid-template-columns (exceto que aplicados a linhas).

Exemplo de Uso:

.grid-container {
    display: grid;
    grid-template-rows: 50px auto 200px; /* Três linhas com alturas específicas */
}

Nome da Propriedade: gap (ou grid-gap)

Descrição:

Define o tamanho das lacunas entre as linhas e colunas em um layout de grade ou flexbox. É um atalho para row-gap e column-gap.

Valores Possíveis:

  • <length>: Um valor de comprimento (ex: 10px, 1em).
  • <percentage>: Um valor percentual.

Exemplos de Uso:

.grid-container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: 20px; /* 20px de espaço entre linhas e colunas */
}

/* Ou usando as propriedades individuais */
.flex-container {
    display: flex;
    row-gap: 10px; /* Espaço entre linhas */
    column-gap: 5px; /* Espaço entre colunas */
}

Modelo de Caixa (Box Model)

width e height

Nome da Propriedade: width

Descrição:

Define a largura da área de conteúdo de um elemento.

Valores Possíveis:

  • <length>: Um valor fixo (ex: 100px, 5em).
  • <percentage>: Uma porcentagem da largura do elemento pai (ex: 50%).
  • auto: Padrão. A largura é determinada automaticamente pelo navegador.
  • inherit: Herda o valor do pai.
  • initial, unset, revert.

Exemplo de Uso:

.my-box {
    width: 300px;
    height: 150px;
}

.responsive-image {
    width: 100%; /* Ocupa a largura total do pai */
    height: auto; /* Mantém a proporção da imagem */
}

Nome da Propriedade: height

Descrição:

Define a altura da área de conteúdo de um elemento.

Valores Possíveis:

  • Mesmos valores que width.
  • Para elementos inline, height não afeta a altura visual; use line-height ou mude para inline-block.

Exemplo de Uso:

.header {
    height: 80px;
    background-color: navy;
}
margin

Nome da Propriedade: margin

Descrição:

Define o espaço externo ao redor de um elemento, criando uma área transparente que separa o elemento de outros elementos. É uma propriedade shorthand para margin-top, margin-right, margin-bottom e margin-left.

Valores Possíveis:

  • <length>: Um valor fixo (ex: 10px, 1em).
  • <percentage>: Uma porcentagem da largura do bloco contendo o elemento.
  • auto: Permite que o navegador calcule a margem. Útil para centralizar blocos (margin: 0 auto;).
  • Pode aceitar 1, 2, 3 ou 4 valores:
    • margin: 10px; (todos os lados)
    • margin: 10px 20px; (top/bottom, left/right)
    • margin: 10px 20px 30px; (top, left/right, bottom)
    • margin: 10px 20px 30px 40px; (top, right, bottom, left)

Exemplos de Uso:

.card {
    margin: 20px; /* 20px em todos os lados */
}

.title {
    margin: 0 auto 10px auto; /* Centraliza horizontalmente, 10px embaixo */
    width: fit-content; /* Necessário para centralizar com margin: auto */
}

.spacing-top {
    margin-top: 50px;
}
padding

Nome da Propriedade: padding

Descrição:

Define o espaço interno entre o conteúdo de um elemento e sua borda. É uma propriedade shorthand para padding-top, padding-right, padding-bottom e padding-left.

Valores Possíveis:

  • <length>, <percentage> (similar a margin).
  • Pode aceitar 1, 2, 3 ou 4 valores (similar a margin).

Exemplos de Uso:

.button {
    padding: 10px 15px; /* 10px top/bottom, 15px left/right */
    background-color: var(--secondary-color);
    color: white;
}

.content-box {
    padding-left: 30px;
    padding-right: 30px;
    border: 1px solid var(--border-color);
}
border

Nome da Propriedade: border

Descrição:

Define a borda ao redor de um elemento. É uma propriedade shorthand para border-width, border-style e border-color.

Valores Possíveis:

  • <border-width>: (ex: 1px, medium, thick).
  • <border-style>: (ex: solid, dashed, dotted, double, groove, ridge, inset, outset, none, hidden).
  • <color>: (ex: red, #000, rgb(255,0,0)).
  • Pode ser aplicado individualmente (ex: border-top, border-right, etc.).

Exemplos de Uso:

.element-with-border {
    border: 2px solid #333;
}

.card-custom-border {
    border-top: 5px dotted blue;
    border-right: 1px solid black;
    border-bottom: none;
}
box-sizing

Nome da Propriedade: box-sizing

Descrição:

Controla como a largura e a altura totais de um elemento são calculadas. Isso inclui ou exclui o padding e a borda na dimensão declarada.

Valores Possíveis:

  • content-box (padrão): A largura e a altura do elemento incluem apenas o conteúdo. Padding e borda são adicionados à dimensão total.
  • border-box: A largura e a altura do elemento incluem o padding e a borda, mas não a margem. Mais intuitivo para layout.

Exemplos de Uso:

/* Padrão para todos os elementos para facilitar o layout */
html {
  box-sizing: border-box;
}
*, *::before, *::after {
  box-sizing: inherit;
}

.my-div {
    width: 200px;
    padding: 10px;
    border: 5px solid black;
    /* Se box-sizing: border-box, a largura real será 200px.
       Se content-box, a largura real será 200px + 2*10px (padding) + 2*5px (border) = 230px */
}

Tipografia

font-family

Nome da Propriedade: font-family

Descrição:

Define a família de fontes a ser usada para um texto. Pode ser uma lista de fontes (font-stack), onde o navegador usa a primeira que encontrar no sistema do usuário.

Valores Possíveis:

  • Nomes de fontes específicos (ex: Arial, "Times New Roman"). Se o nome tiver espaços, use aspas.
  • Nomes genéricos de família de fontes (ex: serif, sans-serif, monospace, cursive, fantasy). É uma boa prática incluir um genérico como fallback.

Exemplos de Uso:

body {
    font-family: Arial, sans-serif;
}

h1 {
    font-family: "Georgia", serif;
}

.code-block {
    font-family: "Courier New", monospace;
}
font-size

Nome da Propriedade: font-size

Descrição:

Define o tamanho da fonte do texto.

Valores Possíveis:

  • <length>: Pixels (16px), em (1em), rem (1rem). em e rem são preferidos para responsividade.
  • <percentage>: Porcentagem do tamanho da fonte do elemento pai (ex: 120%).
  • Palavras-chave absolutas: xx-small, x-small, small, medium, large, x-large, xx-large.
  • Palavras-chave relativas: larger, smaller.

Exemplos de Uso:

p {
    font-size: 16px;
}

h2 {
    font-size: 2em; /* 2 vezes o tamanho da fonte do pai */
}

.responsive-text {
    font-size: 1.2rem; /* 1.2 vezes o tamanho da fonte base do HTML */
}
font-weight

Nome da Propriedade: font-weight

Descrição:

Define a espessura (ou negrito) da fonte. Nem todas as fontes suportam todos os pesos.

Valores Possíveis:

  • Palavras-chave: normal (equivalente a 400), bold (equivalente a 700).
  • Valores numéricos: De 100 a 900 em múltiplos de 100 (ex: 400 para normal, 700 para bold).
  • lighter, bolder: Relativo ao peso do elemento pai.

Exemplos de Uso:

strong {
    font-weight: bold;
}

h3 {
    font-weight: 500; /* Peso intermediário, se a fonte suportar */
}
line-height

Nome da Propriedade: line-height

Descrição:

Define a altura de cada linha de um texto, influenciando o espaçamento vertical entre as linhas.

Valores Possíveis:

  • normal: Valor padrão, geralmente em torno de 1.2.
  • <number>: Um número sem unidade (ex: 1.5), que é um fator multiplicador do font-size atual do elemento. É o mais recomendado.
  • <length>: Um valor fixo (ex: 20px).
  • <percentage>: Uma porcentagem do font-size do elemento.

Exemplos de Uso:

p {
    line-height: 1.6; /* 1.6 vezes o tamanho da fonte */
}

h1 {
    line-height: 1; /* Altura de linha apertada */
}
text-align

Nome da Propriedade: text-align

Descrição:

Especifica o alinhamento horizontal do conteúdo em linha de um elemento de bloco ou célula de tabela.

Valores Possíveis:

  • left: Alinha o texto à esquerda (padrão em LTR).
  • right: Alinha o texto à direita.
  • center: Centraliza o texto.
  • justify: Alinha o texto para que tanto as bordas esquerda quanto direita de cada linha sejam retas (exceto a última linha).

Exemplos de Uso:

.article-body {
    text-align: justify;
}

.footer-text {
    text-align: center;
}
color

Nome da Propriedade: color

Descrição:

Define a cor do primeiro plano do conteúdo textual de um elemento e de quaisquer decorações de texto.

Valores Possíveis:

  • Nomes de cores: red, blue, green, black, white, etc.
  • Hexadecimal: #RRGGBB (ex: #FF0000 para vermelho), #RGB (ex: #F00 para vermelho).
  • RGB/RGBA: rgb(R, G, B) (ex: rgb(255, 0, 0)), rgba(R, G, B, A) (ex: rgba(255, 0, 0, 0.5) para vermelho com 50% de opacidade).
  • HSL/HSLA: hsl(H, S%, L%), hsla(H, S%, L%, A).
  • Palavras-chave: currentColor (cor atual do texto), transparent.

Exemplos de Uso:

p {
    color: #333;
}

.highlight-text {
    color: rgba(0, 123, 255, 0.8);
}

Cores e Fundos

background-color

Nome da Propriedade: background-color

Descrição:

Define a cor de fundo de um elemento. Esta cor preenche a área de conteúdo, padding e borda do elemento.

Valores Possíveis:

  • Qualquer valor de cor válido (nomes, hex, RGB/RGBA, HSL/HSLA).
  • transparent (padrão).

Exemplos de Uso:

.header {
    background-color: #007bff;
    color: white;
}

.light-section {
    background-color: #f8f8f8;
}
background-image

Nome da Propriedade: background-image

Descrição:

Define uma ou mais imagens de fundo para um elemento.

Valores Possíveis:

  • url('path/to/image.jpg'): Caminho para o arquivo da imagem.
  • none (padrão): Nenhuma imagem de fundo.
  • Funções de gradiente: linear-gradient(), radial-gradient(), etc.

Exemplos de Uso:

.hero-banner {
    background-image: url('images/hero.jpg');
    background-size: cover;
    background-position: center;
}

.gradient-background {
    background-image: linear-gradient(to right, blue, green);
}
background-size

Nome da Propriedade: background-size

Descrição:

Define o tamanho das imagens de fundo.

Valores Possíveis:

  • auto (padrão).
  • cover: Redimensiona a imagem para cobrir toda a área de fundo, cortando-a se necessário.
  • contain: Redimensiona a imagem para que caiba completamente dentro da área de fundo, sem cortar, mas pode deixar espaço vazio.
  • <length>: Valores fixos (ex: 100px 50px para largura e altura).
  • <percentage>: Valores percentuais (ex: 50% 100%).

Exemplos de Uso:

.full-bg {
    background-image: url('img.jpg');
    background-size: cover; /* A imagem cobre todo o elemento */
}

.icon-bg {
    background-image: url('icon.png');
    background-size: 20px 20px; /* Tamanho fixo para o ícone */
    background-repeat: no-repeat;
}

Bordas e Contornos

border-radius

Nome da Propriedade: border-radius

Descrição:

Define o arredondamento dos cantos da borda de um elemento. É uma propriedade shorthand que pode arredondar todos os quatro cantos ou cantos individuais.

Valores Possíveis:

  • <length>: Um valor fixo (ex: 5px, 1em).
  • <percentage>: Uma porcentagem da dimensão correspondente do box (largura/altura).
  • Pode aceitar 1, 2, 3 ou 4 valores (para top-left, top-right, bottom-right, bottom-left, respectivamente, se forem 4).
  • Para um círculo perfeito, use 50% em um elemento quadrado.

Exemplos de Uso:

.button {
    border-radius: 5px; /* Arredonda todos os 4 cantos em 5px */
}

.circle-image {
    width: 100px;
    height: 100px;
    border-radius: 50%; /* Transforma o elemento em um círculo/oval */
}

.card-custom-corners {
    border-radius: 10px 0 10px 0; /* Top-left, Top-right, Bottom-right, Bottom-left */
}
outline

Nome da Propriedade: outline

Descrição:

Desenha uma linha ao redor de elementos, *fora* da borda. Não ocupa espaço e não afeta o layout do documento. Usada frequentemente para indicar o foco do teclado para acessibilidade.

Valores Possíveis:

  • Shorthand para outline-width, outline-style, outline-color.
  • Valores similares a border (width, style, color).

Exemplos de Uso:

/* Remove o outline padrão ao focar (use com cuidado, pois afeta a acessibilidade) */
button:focus {
    outline: none;
    box-shadow: 0 0 0 3px rgba(0, 123, 255, 0.5); /* Adiciona um indicador visual alternativo */
}

.highlighted-element {
    outline: 2px dashed green;
    outline-offset: 5px; /* Espaçamento entre a borda e o outline */
}

Efeitos Visuais e Transformações

opacity

Nome da Propriedade: opacity

Descrição:

Define o nível de opacidade de um elemento (a transparência). Afeta o elemento inteiro, incluindo seu conteúdo (texto, imagens).

Valores Possíveis:

  • <number>: Um valor entre 0.0 (totalmente transparente) e 1.0 (totalmente opaco).

Exemplos de Uso:

.faded-image {
    opacity: 0.7; /* 70% opacidade */
}

.hidden-element-with-opacity {
    opacity: 0; /* Elemento invisível, mas ainda ocupa espaço */
}

.hover-effect:hover {
    opacity: 0.8;
}
box-shadow

Nome da Propriedade: box-shadow

Descrição:

Adiciona uma ou mais sombras a um elemento. As sombras podem ser internas ou externas.

Valores Possíveis:

  • none (padrão).
  • inset (opcional): Torna a sombra interna.
  • offset-x offset-y: Deslocamento horizontal e vertical da sombra (obrigatórios).
  • blur-radius (opcional): Quanto maior, mais borrada a sombra.
  • spread-radius (opcional): Expande ou encolhe a sombra.
  • color: Cor da sombra.
  • Podem ser definidas múltiplas sombras separadas por vírgulas.

Exemplos de Uso:

.card {
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3); /* Sombra simples */
}

.inset-shadow {
    box-shadow: inset 0 0 10px red; /* Sombra interna vermelha */
}

.multiple-shadows {
    box-shadow: 0 4px 8px rgba(0,0,0,0.2), 0 6px 20px rgba(0,0,0,0.19);
}
text-shadow

Nome da Propriedade: text-shadow

Descrição:

Adiciona uma ou mais sombras ao texto de um elemento.

Valores Possíveis:

  • none (padrão).
  • offset-x offset-y: Deslocamento horizontal e vertical da sombra (obrigatórios).
  • blur-radius (opcional): Quanto maior, mais borrada a sombra.
  • color: Cor da sombra.
  • Podem ser definidas múltiplas sombras separadas por vírgulas.

Exemplos de Uso:

h1 {
    color: white;
    text-shadow: 2px 2px 4px black; /* Sombra preta para o texto */
}

.neon-text {
    color: white;
    text-shadow:
        0 0 5px #fff,
        0 0 10px #fff,
        0 0 20px #ff00ff,
        0 0 30px #ff00ff,
        0 0 40px #ff00ff,
        0 0 50px #ff00ff,
        0 0 70px #ff00ff;
}
transform

Nome da Propriedade: transform

Descrição:

Aplica transformações 2D ou 3D a um elemento, como translação (movimento), rotação, escala e inclinação. Geralmente usado com transition para animações suaves.

Valores Possíveis:

  • Funções 2D:
    • translate(x, y): Move o elemento.
    • translateX(x), translateY(y).
    • rotate(angle): Rotaciona o elemento.
    • scale(x, y): Redimensiona o elemento.
    • scaleX(x), scaleY(y).
    • skew(x-angle, y-angle): Inclina o elemento.
  • Funções 3D: translate3d(), rotateX(), rotateY(), rotateZ(), scale3d(), etc.
  • none (padrão).

Exemplos de Uso:

.square {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: transform 0.3s ease-in-out;
}

.square:hover {
    transform: rotate(45deg) scale(1.2); /* Rotaciona e aumenta */
}

.button-press:active {
    transform: translateY(2px); /* Efeito de botão pressionado */
}

Transições e Animações

transition

Nome da Propriedade: transition

Descrição:

Permite que você defina como as mudanças nas propriedades CSS devem acontecer ao longo do tempo, em vez de serem instantâneas. É uma propriedade shorthand para transition-property, transition-duration, transition-timing-function e transition-delay.

Valores Possíveis:

  • <property>: A propriedade CSS a ser animada (ex: opacity, transform, all para todas).
  • <duration>: Duração da transição (ex: 0.5s, 500ms).
  • <timing-function>: Curva de velocidade da transição (ex: ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier()).
  • <delay>: Atraso antes do início da transição (ex: 0.2s).
  • Podem ser definidas múltiplas transições separadas por vírgulas.

Exemplos de Uso:

.fade-in-out {
    opacity: 0;
    transition: opacity 1s ease-in-out; /* Transiciona a opacidade em 1 segundo */
}

.fade-in-out.show {
    opacity: 1;
}

.menu-item {
    padding: 10px;
    background-color: lightgray;
    transition: background-color 0.3s, transform 0.2s; /* Múltiplas propriedades */
}

.menu-item:hover {
    background-color: gray;
    transform: translateX(5px);
}
animation (e @keyframes)

Nome da Propriedade: animation

Descrição:

A propriedade animation é uma shorthand para várias subpropriedades de animação CSS. Permite que você defina uma sequência de estilos CSS que serão aplicados a um elemento ao longo do tempo. As animações são definidas com a regra @keyframes.

Valores Possíveis:

  • <animation-name>: Nome da animação definida com @keyframes.
  • <animation-duration>: Duração de um ciclo da animação.
  • <animation-timing-function>: Curva de velocidade.
  • <animation-delay>: Atraso antes do início.
  • <animation-iteration-count>: Quantas vezes a animação deve se repetir (número ou infinite).
  • <animation-direction>: Se a animação deve tocar para frente, para trás ou alternar (normal, reverse, alternate, alternate-reverse).
  • <animation-fill-mode>: Estilos aplicados antes/depois da animação (none, forwards, backwards, both).
  • <animation-play-state>: Se a animação está rodando ou pausada (running, paused).

Exemplos de Uso (com @keyframes):

/* Define a animação */
@keyframes slidein {
  from {
    transform: translateX(0%);
    opacity: 0;
  }
  to {
    transform: translateX(100%);
    opacity: 1;
  }
}

/* Aplica a animação a um elemento */
.animated-box {
  width: 100px;
  height: 100px;
  background-color: orange;
  animation-name: slidein;
  animation-duration: 3s;
  animation-iteration-count: infinite; /* Repete indefinidamente */
  animation-direction: alternate; /* Alterna a direção a cada ciclo */
  animation-timing-function: ease-in-out;
}

/* Shorthand */
.shorthand-box {
  animation: slidein 3s ease-in-out infinite alternate;
}
